# Blogspoint

It is a full stack blog website. One can login, register easily and then read any blog available on the site.  




## Tech Stack

**Client:** React JS, ChakraUI

**Server:** Node JS, Express JS

**Database:** Mongo DB 

**Other:** Redux, Cloudinary
